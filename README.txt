MERT DÜZEL HESAPLAMA UYGULAMASI
=====================================

PROJE ADI:
Mert DÜZEL Hesaplama Uygulaması

PROJE KONUSU:
JavaScript ile Çok Fonksiyonlu Hesaplama Aracı

PROJE AMACI:
Bu proje, JavaScript kullanılarak Hesaplama.net benzeri çok fonksiyonlu ve kullanıcı dostu bir web uygulaması geliştirmek amacıyla hazırlanmıştır.
Uygulama; matematik, geometri, finans ve sağlık alanlarında farklı hesaplama araçlarını tek bir sistem içerisinde sunmaktadır.
Proje responsive (mobil uyumlu) tasarıma sahiptir ve farklı cihazlarda düzgün çalışacak şekilde hazırlanmıştır.

KULLANILAN TEKNOLOJİLER:
- HTML5
- CSS3
- JavaScript (Vanilla JS)

PROJE DOSYA YAPISI:
- index.html        -> Ana sayfa
- matematik.html    -> Matematik hesaplamaları
- geometri.html     -> Geometri hesaplamaları
- finans.html       -> Finans hesaplamaları
- saglik.html       -> Sağlık hesaplamaları
- style.css         -> Tüm sayfaların tasarım dosyası
- script.js         -> Hesaplama fonksiyonlarının bulunduğu JavaScript dosyası
- README.txt        -> Proje açıklama ve kullanım dokümanı

UYGULAMA ÖZELLİKLERİ:
Bu projede toplam 20 farklı hesaplama özelliği bulunmaktadır.

PROJENİN ÇALIŞTIRILMASI:
1. Tüm dosyaları aynı klasör içerisinde tutunuz.
2. index.html dosyasını bir internet tarayıcısında açınız.
3. Ana sayfadan ilgili kategori seçilerek hesaplama sayfalarına geçiş yapılabilir.
4. Her sayfadaki hesaplama kutularına değerler girilerek sonuç alınabilir.

RESPONSIVE (MOBİL UYUMLULUK):
Proje responsive tasarım prensibine uygun olarak geliştirilmiştir.
CSS medya sorguları (media queries) kullanılarak mobil telefon, tablet ve masaüstü cihazlarda düzgün görünmesi sağlanmıştır.

TEST VE DOĞRULUK KONTROLLERİ:
Uygulamadaki hesaplama fonksiyonları farklı örnek değerler ile test edilmiştir.
Ayrıca hata kontrolü gereken alanlarda gerekli önlemler alınmıştır.

Test edilen örnek durumlar:
- Bölme işleminde 0'a bölme kontrolü yapılmıştır.
- Karekök işleminde negatif sayı kontrolü yapılmıştır.
- Faktöriyel işleminde negatif ve tam sayı kontrolü yapılmıştır.
- BMI hesaplamasında boy değerinin 0'dan büyük olması kontrol edilmiştir.
- Yaş hesaplamasında geçerli doğum yılı kontrolü yapılmıştır.
- Hız hesaplamasında zaman değerinin 0 olmaması kontrol edilmiştir.
